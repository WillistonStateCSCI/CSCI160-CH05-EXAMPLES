# CSCI160-CH05-EXAMPLES
Programming Examples From Java Illuminated Book by Anderson and Franceschi  
**Example05_01** - A demonstration of logical operators at work  
**Example05_02** - Using *if* statements to calculate a final test grade  
**Example05_03** - Using *if/else* statements to determine a passing grade  
**Example05_04** - Using *if/else if* to translate a numeric grade into a letter grade  
**Example05_05** - Finding the smallest of three integers with sequential *if* statements  
**Example05_06** - Guessing a number between 1 and 10 with nested *if* statements  
**Example05_08** - A demonstration of equality operators on Floating-Point Numbers  
**Example05_09** - Using a threshold to compare floating-point numbers  
**Example05_10** - Using BigDecimal to computer precise decimal numbers  
**Example05_11** - Comparing objects using the equality operator and then using the *equals* method  
**Example05_12** - Using *equals*, *equalsIgnoreCase* and *compareTo* to compare string objects  
**Example05_13** - A demonstration of the conditional operator *?* to shorten code  
**Example05_14** - A combination of concepts used together to create a simple calculator
